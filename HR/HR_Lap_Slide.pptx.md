# Introduction
- Large Language Models (LLMs) such as ChatGPT can support HR departments by generating structured content, analyzing text, and improving workflow efficiency.
- This lecture covers:
  - Performance review design
  - Competency framework development
  - Culture audits
  - Three hands-on lab sessions for students

---

# How HR Departments Use LLMs
- Recruitment and Job Descriptions
- Interview Questions and Rubrics
- Learning and Development
- Performance Management
- HR Policies and Documentation
- Employee Feedback Analysis
- Administrative Support

---

# Recruitment Support with LLMs
- Generate job descriptions tailored to specific roles
- Extract relevant skills from resumes
- Create interview questions
- Draft evaluation rubrics
- Analyze candidate fit based on keywords and competencies

Example prompt:
"Create a job description for a Marketing Manager including role summary, responsibilities, required skills, and preferred qualifications."

---

# Learning and Development Support
- Create training modules
- Summarize textbooks or best practices
- Generate quizzes or case studies
- Recommend competency-based learning paths

Example:
"Write a training outline for new supervisors focusing on delegation and communication skills."

---

# Performance Management with LLMs
- Draft performance review templates
- Suggest performance indicators
- Rewrite feedback in a professional tone
- Align behaviors with competency standards

Example:
"Write constructive feedback for an employee improving in teamwork but needing better time management."

---

# HR Policy Development
- Draft new policies
- Simplify existing documents
- Translate tone for different audiences

Example:
"Rewrite this HR policy in clear and friendly language for new employees."

---

# Culture Audit Support
- Identify themes in employee feedback
- Summarize organizational strengths and weaknesses
- Suggest culture improvement actions
- Map findings to leadership behavior

Example:
"Conduct a culture audit for a company facing communication challenges and low trust."

# Designing Performance Reviews
- Key components of a good performance review:
  - Key performance areas (KPAs)
  - Behavioral indicators
  - Rating scale
  - Developmental feedback
  - Employee self-reflection

Example prompt:
"Create a performance review template for a mid-level manager with KPAs, rating scales, and development questions."

Example prompt:
"Create a competency framework for mid-level managers including definitions and behavioral indicators."

# Lab Session 1: Generate a Job Description

## Objective
Learn how prompt quality affects job description output.

## Instructions
1. Select a job role (HR Assistant, Data Analyst, Marketing Coordinator, etc.)
2. Include the following in your prompt:
   - Job title
   - Industry or company type
   - Required skills and responsibilities
   - Preferred qualifications
   - Output structure (with headings)

Example prompt:
"Write a job description for a Data Analyst in a technology company including role summary, responsibilities, required skills, preferred qualifications, and tools used."

---

# Lab Session 1 Deliverables
- Final prompt created by the student
- Generated job description
- Short reflection (5–6 sentences) on:
  - Strengths of the output
  - Weaknesses of the initial attempt
  - How revising the prompt improved the output

---

# Lab Session 2: Interview Questions and Evaluation Rubric

## Objective
Use LLMs to create structured interview materials.

## Instructions
1. Use the same job role from Lab 1.
2. Create a prompt asking for:
   - Five behavioral interview questions
   - Five technical or job-related questions
   - A three-level evaluation rubric
   - Indicators for what a strong answer includes

Example prompt:
"Generate interview questions for a Marketing Manager including behavioral and technical questions, a rubric, and strong answer indicators."

---

# Lab Session 2 Deliverables
- Behavioral interview questions
- Technical interview questions
- Three-level evaluation rubric
- Reflection on which questions were most effective and why

---

# Lab Session 3: Employee Feedback Analysis

## Objective
Analyze sample employee comments using LLM capabilities.

## Instructions
Use the following sample comments:
- Communication between departments is slow.
- My manager gives helpful feedback.
- Workload has increased significantly.
- I appreciate the remote work policy.
- Career development opportunities are unclear.
- Meetings run too long.

Students must ask the LLM to:
- Categorize each comment
- Summarize key insights
- Recommend HR actions
- Present results in a table

Example prompt:
"Analyze these employee comments, categorize them, summarize key insights, recommend three HR actions, and create a table with comment, category, and action."

---

# Lab Session 3 Deliverables
- Categorized comments
- Summary insights (4–5 sentences)
- Three HR recommendations
- Table containing analysis
- Reflection on the quality and limitations of the LLM analysis
